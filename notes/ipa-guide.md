TEPS 단어 옆에 붙는 `/ˈvɪdʒɪlənt/` 같은 기호가 IPA(국제 음성 기호)입니다.
처음엔 낯설지만 패턴을 알면 금방 익힐 수 있어요.

---

## 자주 나오는 모음

| 기호 | 한글 근사 발음 | 예시 단어 |
|------|--------------|-----------|
| iː   | 이~ (길게)    | see /siː/ |
| ɪ    | 이 (짧게)     | sit /sɪt/ |
| e    | 에            | bed /bed/ |
| æ    | 애 (입 크게)  | cat /kæt/ |
| ʌ    | 어 (짧게)     | cup /kʌp/ |
| ɑː   | 아~ (길게)    | car /kɑːr/ |
| ɒ    | 오 (영국식)   | hot /hɒt/ |
| ɔː   | 오~ (길게)    | law /lɔː/  |
| uː   | 우~ (길게)    | food /fuːd/ |
| ʊ    | 우 (짧게)     | book /bʊk/ |
| ə    | 어 (약하게, 강세 없는 음절) | about /əˈbaʊt/ |
| eɪ   | 에이           | day /deɪ/ |
| aɪ   | 아이           | my /maɪ/  |
| ɔɪ   | 오이           | boy /bɔɪ/ |
| aʊ   | 아우           | now /naʊ/ |
| əʊ   | 오우           | go /ɡəʊ/  |

---

## 자주 나오는 자음

| 기호 | 설명 | 예시 |
|------|------|------|
| θ    | th (혀를 이 사이에) 무성음 | think /θɪŋk/ |
| ð    | th 유성음 | this /ðɪs/ |
| ʃ    | 쉬 | she /ʃiː/ |
| ʒ    | 쥐 (부드럽게) | measure /ˈmeʒər/ |
| tʃ   | 치 | church /tʃɜːrtʃ/ |
| dʒ   | 지 | judge /dʒʌdʒ/ |
| ŋ    | 받침 -ㅇ | sing /sɪŋ/ |
| r    | 혀를 말아서 | red /red/ |

---

## 강세 기호

- **ˈ** (위 어깻표) — 바로 다음 음절에 강세. 예: `ˈvɪdʒɪlənt` → **VI**-gi-lant
- **ˌ** (아래 어깻표) — 약한 강세 (2차 강세)

강세 있는 음절을 살짝 더 크고 길게 읽으면 됩니다.

## 읽는 연습 순서 추천
